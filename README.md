# 🚀 React Modern Todo App

Bu proje, React ve Tailwind CSS kullanılarak geliştirilmiş, modern arayüzlü ve kullanıcı dostu bir yapılacaklar listesi (Todo List) uygulamasıdır. Proje; görev ekleme, silme, düzenleme ve ilerleme takibi gibi temel özelliklere odaklanırken, estetik bir kullanıcı deneyimi sunmayı hedefler.

## ✨ Özellikler

-   **🎯 Görev Yönetimi:** Yeni görevler ekleyin, tamamlananları işaretleyin veya silin.
-   **✏️ Düzenleme Modu:** Mevcut görevlerin başlıklarını anında güncelleyin.
-   **📊 İlerleme Takibi:** Tamamlanan görev oranına göre dinamik olarak güncellenen istatistik kartları ve ilerleme çubuğu (Progress Bar).
-   **🎨 Modern Arayüz:** Tailwind CSS ile hazırlanan Glassmorphism efektleri, yumuşak geçişler ve tamamen responsive (mobil uyumlu) tasarım.
-   **⏳ Yükleme Durumları:** İşlemler sırasında kullanıcıya geri bildirim veren loading animasyonları.

## 🛠️ Kullanılan Teknolojiler

* **Framework:** [React](https://reactjs.org/)
* **Derleyici:** [Vite](https://vitejs.dev/)
* **Stil:** [Tailwind CSS](https://tailwindcss.com/)
* **İkonlar:** Heroicons / Custom SVG

## 🚀 Kurulum

Projeyi yerel bilgisayarınızda çalıştırmak için aşağıdaki adımları izleyebilirsiniz:

1.  **Depoyu Klonlayın:**
    ```bash
    git clone [https://github.com/kullanici-adin/repo-adin.git](https://github.com/kullanici-adin/repo-adin.git)
    ```

2.  **Proje Dizinine Gidin:**
    ```bash
    cd repo-adin
    ```

3.  **Gerekli Paketleri Yükleyin:**
    ```bash
    npm install
    ```

4.  **Projeyi Başlatın:**
    ```bash
    npm run dev
    ```
    *Uygulama varsayılan olarak `http://localhost:5173` adresinde çalışacaktır.*

## 📦 Yayına Alma (Deployment)

Bu proje **Netlify** üzerinde yayına alınmıştır. Her yeni `push` işleminde otomatik olarak derlenmekte ve güncellenmektedir.
